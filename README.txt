Proyecto No.4: JUEGO DE PREGUNTAS Y RESPUESTAS
Hecho por: Valentín Enrique Rodríguez Fausto
Grupo: 732
Integrantes: 1 (Yo)

El proyecto en cuestión es un juego simple de preguntas y respuestas, donde el programa le ofrece mediante un MENU al usuario
una opción a elegir para comenzar el juego, y otra opción por si decide salir para dejar el programa.

IMPLEMENTACIONES:
    1. Lenguaje C para imprimir preguntas y cargar respuestas.
    2. Lenguaje ASM para el calculo de aciertos mediante comparativa de respuestas correctas con respuestas de usuario.
    3. Makefile como compilador -lncurses para su debido funcionamiento aparte de un -no-pie por requisito igualmente.
    4. ncurses utilizado para lenguaje C, para la creación de interfaces.


COMO SE CONFIGURA:
    No se realmente a que se refiere con esto mismo, sin embargo los unicos requisitos son realmente los minimos para poder utilizar github
    o en caso de querer utilizarlo en tu propia PC, entonces los requisitos serian los minimos para poder utilizar UBUNTU y WSL, etc.
    de formar correcta.

COMO UTILIZARLO:
    Esto es bastante sencillo de utilizar, al ser un programa que realmente no requiere mucho para poder ejecutarse, por ende las insutrucciones
    a seguir son cortas y directas.

    Instrucciones:
        1. Abrir Ubuntu
        2. Localizar mediante "cd" la carpeta en la que se guarda el repositorio en caso de haberlo descargado
        3. Utilizar "make release" para la creación del ejecutable.
        4. Utilizar "make run" para iniciar.
        5. Simplemente elige sus opciones y diviertete.
